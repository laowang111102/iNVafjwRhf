# 超市管理系统 java50

## 项目概述

本项目是基于Spring Boot和Vue的前后端分离的超市管理系统，主要用于实现超市的进销存管理，提高超市供销存管理的效率。

## 技术栈

- 主要编程语言：Java
- 后端框架：Spring Boot
- 数据持久层框架：MyBatis-Plus
- 缓存技术：Redis
- 前端框架：Vue
- 开发工具和环境：
  - IntelliJ IDEA
  - JDK 1.8
  - Maven
  - Node.js
  - HBuilder

## 功能模块

- 用户管理
- 订单管理
- 地址管理
- 供应商管理

## 系统角色

系统提供管理员角色，管理员可以自由设置并分配权限。

## 运行环境

- 数据库：MySQL

## 核心亮点

- 前后端分离，便于维护和扩展。
- 采用成熟稳定的Spring Boot和Vue技术栈，保障系统性能。
- 使用Redis进行缓存，提升系统响应速度。
- 系统支持二次开发，满足不同定制化需求。
